# Get Agreement confidence interval

Get Agreement confidence interval

## Usage

``` r
get_ci(X, CONFIDENCE = 0.95)
```

## Arguments

- X:

  Object fitted with
  [`agreement()`](https://giuseppealfonzetti.github.io/AgreementPhi/reference/agreement.md)
  function.

- CONFIDENCE:

  Confidence level.

## Value

Returns a list with estimate, standard error and confidence interval
